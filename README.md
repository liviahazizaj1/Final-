# 4.System features

# 4.1 Log in System

Log In System is one of our critical components in Pharmaget , ensuring secure access to authorized users.
It does involve a user authentication process where users provide with their credentials which is : Username and Password to gain access to their Pharmaget account.
Some of the features in the Log in System are : the user authentication which validates the credentials that match with the stores database entries.
Password encryption which ensures passwords that are securely stored using the encryption techniques.
User Roles which differentiate roles as costumer or admin to provide role-based control.Admins will be able to change all types of products in the system and check all orders of the costumers , meanwhile costumers will be able to just check the products that they ordered.
# 4.2 Products Ordered by Costumers

This feature of the system allows the client to browse , select and order the products that are available in the website.
It includes functionality to add products to the card , check the products and make purchases.
The product catalog will help the costumers to check all the products that Pharmaget has to offer , and with the products name , description and price.
Searching the product that they need will be easier when the costumer is allowed to filter regarding the product that is needed.
And the last one is the Checkout process , which after checking the products ,this process makes sure that the purchase is complete and will be sent to the shipping information that the client has given.

# 4.3 Getting Doctors Note and Prescription

This is a feature that allows clients to upload the doctors notes or prescription when ordering the products.
This will make sure that the pharmacy verifies the prescription in the best way before sending the medicine .
The feature is called “Upload prescription” which allows the client to upload an image or PDF of their doctors note.
And the other feature is the “Prescription Verification” that allows our staff to review and verify the uploaded prescription.
# 4.4 Adding and Deleting Products

This feature is mostly for admin users that allows them to add or delete the products based on their stock and new products in the store.
“Add Product” field is where the admin can fill the form for a new product such as name , description , category and price.
“Delete Product” field is where authorized admin users remove the products from their catalog based on their stock.
“Edit Products” field is where the admin can change the feature of existing products on web.
